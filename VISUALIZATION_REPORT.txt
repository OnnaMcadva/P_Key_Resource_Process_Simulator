VISUALIZATION COMPONENTS


1. SimulationVisualizer.java (5.2KB)
    Type: Main JFrame window
    Size: 1400x850 pixels
    Interface: Tabbed pane with 3 tabs
    Framework: Java Swing (javax.swing)
    Key Features:
       JTabbedPane for tab management
       GanttChartPanel integration
       ResourceGraphPanel integration
       Summary statistics display
       BorderLayout for organization

2. GanttChartPanel.java (6.9KB)
    Type: JPanel (graphics component)
    Rendering: Custom paintComponent
    Data: ProcessExecution array
    Features:
       Timeline rendering (X-axis: time)
       Process bars (Y-axis: processes)
       Color coding for process types
       Start/end time visualization
       Process dependency indication
       Hover/click event handling

3. ResourceGraphPanel.java (8.8KB)
    Type: JPanel (graphics component)
    Rendering: Custom paintComponent
    Data: ResourceSnapshot array
    Features:
       Multi-line graph rendering
       Resource tracking over time
       Color coding per resource
       Y-axis scaling
       Time-based X-axis
       Real-time evolution display


ARCHITECTURE


Entry Point: Krpsim.java (main class)
 Command-line: --visualize flag
 OptimizationStrategy.optimize()  OptimizationResult
 SimulationVisualizer.show(result, config, maxDelay)
   Create JFrame instance
   Initialize three panels:
     GanttChartPanel (process timeline)
     ResourceGraphPanel (evolution graphs)
     Summary Panel (statistics)
   Display window
 Block until user closes window

Data Flow:
OptimizationResult  Visualizer  Panels  Graphics2D  Screen
